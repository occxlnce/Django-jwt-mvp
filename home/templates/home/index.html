Django REST API with JWT Authentication

This is a small Django REST API platform built for an interview test, implementing user authentication with JWT, role-based access control, and secured resource endpoints.

Available Links:

  • Django Admin Site - Manage users and resources.
  • Swagger UI Documentation - Interactive API documentation.
  • ReDoc Documentation - Alternative API documentation format.
  • API Registration Endpoint - Use POST to register new users.
  • API Token Obtain Endpoint - Use POST to get JWT tokens.
  • API Token Refresh Endpoint - Use POST to refresh JWT tokens.
  • API User Info Endpoint - Use GET with Bearer token to get logged-in user info.
  • API Resources Endpoint - Access protected resources (permissions apply).

Testing

Postman was used to test the API endpoints and permissions cURL/Invoke-WebRequest. Refer to the README.md file for setup instructions and sample requests.
